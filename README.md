## 🚀 Deployment Instructions  
### **Using GitHub Pages**  
1. Upload your files to a **public GitHub repository**.  
2. Go to **Settings → Pages** → Enable **GitHub Pages**.  
3. Select the main branch and save.  
4. Your site will be live at: `https://yourusername.github.io/your-repo/`.  

### **Using Vercel**  
1. Sign up at **[vercel.com](https://vercel.com)**.  
2. Import your **GitHub repository**.  
3. Deploy with default settings (Static Site).  
4. Get your **live portfolio URL**!
