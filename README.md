# Broken Vzn Portfolio

![Portfolio Preview](https://files.catbox.moe/la0rsq.jpg) <!-- Replace with actual screenshot if available -->

A sleek, responsive personal portfolio website for **Broken Vzn**, showcasing full-stack development projects, creative writing, and online presence. Built with modern web technologies for a glassmorphism-inspired design, dynamic backgrounds, and smooth interactions. Perfect for developers and writers who want a professional yet artistic online hub.

## üöÄ Features

- **Responsive Design**: Fully mobile and web responsive, adapting seamlessly across devices (phones, tablets, desktops).
- **Theme Toggle**: Switch between light and dark modes with local storage persistence.
- **Interactive Elements**:
  - Parallax cloud effects on mouse/touch.
  - Rotating background images (fetched from Waifu.pics API with fallbacks).
  - Tabbed navigation for sections (About, Work, Channels, Writing, 3D Card, Contact).
  - Horizontal carousel for work projects (scrollable cards).
- **Profile Highlights**:
  - Expanded profile card with recent updates, GitHub repos, and featured images.
  - Social media integrations (Discord, NightCafe, X/Twitter, Facebook, WhatsApp channels).
- **Contact Form**: Simulated form with WhatsApp quick links (integrate with your backend for real submissions).
- **Accessibility**: Keyboard navigation, ARIA labels, and smooth scrolling.

## üõ† Tech Stack

- **Frontend**: HTML5, Tailwind CSS (via CDN), Vanilla JavaScript.
- **Styling**: Custom CSS for glassmorphism, gradients, and animations (Poppins font).
- **APIs/Integrations**:
  - Waifu.pics for dynamic backgrounds.
  - GitHub for repo links.
- **Deployment**: Static site ‚Äì host on GitHub Pages, Vercel, Netlify, or any static host.
- **No Build Tools**: Single-file setup for quick previews.

## üîß Quick Start

1. **Clone or Download**:
   ```
   git clone <your-repo-url>
   ```
   Or download the `index.html` file directly.

2. **Open in Browser**:
   - Double-click `index.html` to open in your default browser.
   - No server required ‚Äì it's fully static!

3. **Customize**:
   - Edit the HTML for personal details (e.g., links, bio).
   - Replace images/avatars with your own.
   - Update GitHub repo links in the profile card.

4. **Deploy**:
   - **GitHub Pages**: Push to a repo and enable Pages in settings.
